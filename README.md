<h1> Вебзастосування "Список завдань" </h1>
<p> Вебзастосування "Список завдань" дає змогу вести облік виконання наявних завдань, оперуючи ними на одній сторінці. У вебзастосуванні реалізовано можливість добавляти нові завдання. Вебзастосування створено за допомогою НТМL, CSS та JavaScript.</p>
<br>
<p> Для перегляду вебзастосування локально послідовно виконайте наступні команди:</p>
<ol>
  <li> <code> git clone https://github.com/yv-ly/app-task-list-javascript.git </code>, щоб клонувати репозиторій на локальний комп’ютер; </li>
  <li> <code> cd app-task-list-javascript </code>, щоб перейти в папку app-task-list-javascript; </li>
  <li> <code> start index.html </code>, щоб переглянути вебзастосування у браузері. </li>
</ol>

## 🛠️ Локальний запуск

Для локального перегляду проєкту необхідно виконати наступні кроки:

1.  **Клонувати репозиторій:**
    ```bash
    git clone [https://github.com/yv-ly/app-task-list-javascript.git](https://github.com/yv-ly/app-task-list-javascript.git)
    ```

2.  **Перейти у папку проєкту:**
    ```bash
    cd app-task-list-javascript
    ```

3.  **Відкрити файл index.html у браузері:**
    ```bash
    start index.html
    ```

## 📦 Розгортання (CI/CD)

Проєкт використовує **GitHub Actions** для автоматичного розгортання на GitHub Pages.
Кожен `push` у гілку `main` ініціює робочий процес, який оновлює версію застосунку.
Конфігурація знаходиться у файлі: `.github/workflows/deploy.yml`.
